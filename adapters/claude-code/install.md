# Claude Code Adapter Install

1. Add the contents of `CLAUDE.snippet.md` to the target project's `CLAUDE.md`.
2. Optionally copy files from `commands/` into the project's `.claude/commands/` directory.
3. Run `memory-custodian init --project-root <project> --with-claude` if the CLI is installed.

Keep `CLAUDE.md` as an entry point. Store durable memory in `docs/memory/`.
