# NightNotes Video Demo Fixture

This directory contains the NightNotes demo project used in the MemoryCustodian video demonstration.

## NightNotes Overview

NightNotes is a small local Python CLI application for managing session notes.

- `nightnotes/`: Python package implementing note storage and CLI interface.
- `tests/`: Unit tests verifying application requirements (such as data persistence across runs).
- `docs/memory/`: MemoryCustodian project memory files (`manifest.md`, `brief.md`, `decisions.md`, `constraints.md`, `do-not-use.md`, `inbox.md`).
- `AGENTS.md`: Agent entry point pointing to MemoryCustodian protocol.
